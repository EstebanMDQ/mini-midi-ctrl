# Hardware

PCB design files and schematics for the mini MIDI controller.

## Structure

- `schematics/` - Circuit schematics (SVG, PDF, source files)
- `pcb/` - PCB layout files (KiCad, Eagle, etc.)
- `gerbers/` - Production-ready Gerber files for manufacturing

## Specifications

See main README.md for complete hardware specifications and BOM.

Target: ≤ 25 × 25 mm PCB, ≤ 8 mm total height
